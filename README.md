## 🚀 Features
- **Full CRUD Operations:** Seamlessly add new contacts, update existing details, and delete outdated entries.
- **Organized Data Management:** Contacts are securely stored and easily retrievable from the database.
- **Simple UI:** A distraction-free, intuitive interface focused entirely on functionality and ease of use.
- **Full-Stack Architecture:** Separated frontend and backend environments for better scalability and maintainability.

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js *(Environment)*
- **Database:** MongoDB

## 🛑 Prerequisites
Ensure you have the following installed on your local machine before running the application:

- A [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) account or a local MongoDB instance.

## ⚙️ How to Run (Step-by-Step)

### 1️⃣ Clone the Repository
Open your terminal and clone the project to your local machine:
```bash
git clone [https://github.com/jeel-detroja/contact-management-system.git](https://github.com/jeel-detroja/contact-management-system.git)
cd contact-management-system
```

### 2️⃣ Set Up the Backend
Open a terminal, navigate to the backend directory, install dependencies, and start the server:
```bash
cd backend
npm install
```
*(Note: Create a `.env` file in the backend folder and add your `MONGODB_URI` connection string before starting the server).*
```bash
npm start
```

### 3️⃣ Set Up the Frontend
Open a **new** terminal window, navigate to the frontend directory, install dependencies, and launch the app:
```bash
cd frontend
npm install
npm start
```

*(Your frontend application should now open automatically in your browser, typically at `http://localhost:3000`)*
